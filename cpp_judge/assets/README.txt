C++ Judge - GUI Version
========================

How to use:
1. Select a problem from the dropdown menu in the top-right
2. Read the problem statement in the "Problem" tab
3. Switch to "Code Editor" tab
4. Choose one of these options:
   - Write your code directly in the editor
   - Click "Open File" to load a .cpp file
   - Drag and drop a .cpp file anywhere on the window
5. Click the "Submit" button (bottom-right)
6. View results in the "Results" tab

Features:
• Syntax highlighting for C++ code
• Live test results with detailed feedback
• Drag-and-drop file support
• Problem descriptions with examples
• Modern, easy-to-use interface

Problems:
1. A. Holiday Of Equality
2. A. Odd Set
3. A. Plus or Minus

Good luck with your submissions!
